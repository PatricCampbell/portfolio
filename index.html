I am Patric Campbell, a full stack web developer based in New York City.

About Me

My passion for technology started when I was 4 years old and got my first computer, a 386. I even had the internet though I mostly used it to look up the weather in Siberia and the Australian Outback. Ever since then I've been interested in all types of tech, going to the University of Pittsburgh for a Bachelors in Information Science and Biology.

I then started working in tech support and systems administration until I received the opportunity to make a few websites. This ignited my passion for web development and programming that led me to learning Ruby on Rails, JavaScript, React, and Redux. I'm looking to further hone my skills and love working with curious, creative people

Recent Work

Glitch

Glitch is a real time chatting web application built using PostgresSQL as its database, Ruby on Rails as the backend, and React with Redux on the frontend. Glitch is inspired by Slack and allows users to chat in public channels as well as two person and multi-person direct messages.

Kilobyte

Kilobyte is a browser game based off of 1024. It is written in vanilla JavaScript, CSS, and HTML.

pMoney

pMoney is a DOM manipulation library inspired by jQuery written in vanilla JavaScript. There is a demo project called FoodMe that utilizes pMoney to make AJAX calls to an api that serves up 10 random rcipes.

YelpCamp

YelpCamp is a full stack web application inspired by Yelp. The backend is written in NodeJS with ExpressJS and a MongoDB database. It utilizies RESTful principles for its API and the frontend uses pug templates.
